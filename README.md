# Project README

## Overview

This project is currently built using a modern technology stack focused on performance and scalability. The system uses:

* **.NET Framework Core** for application logic
* **TypeScript** for type-safe development
* **Elasticsearch** for powerful search and indexing

The architecture is designed to stay flexible so more technologies can be added later.

---

## Tech Stack

### Application Layer

* .NET Framework (Non-Core)
* HTML
* CSS
* JavaScript/TypeScript

### Search Engine

* Elasticsearch

  * Fast and scalable search
  * Efficient indexing and querying of large datasets
